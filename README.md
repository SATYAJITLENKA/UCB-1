# United Colors of Benetton: P365 Polo Campaign Landing Page

This is a premium, interactive web landing page designed and built for the **P365 Polo Campaign** by United Colors of Benetton. 

Centering around the theme: **"The Chameleon - Nature's Master of Effortless Adaptability"**, the landing page showcases the P365 as the ultimate versatile wardrobe essential.

## Key Features

1. **Reactive Hero Section:** Displaying the P365 branding, slogan ("Crafted to belong. Designed to stand apart."), and a shifting chameleon profile card that dynamically syncs with your selected customizer colors.
2. **Interactive Polo Customizer:** Let users toggle between H/S (Half Sleeve) and F/S (Full Sleeve) options, and choose from the 9 official campaign colorways (e.g. Deep Green, Pastel Wheat, Deep Teal, Deep Red with Gold tipping) to see the polo change colors live.
3. **Product Truths (USPs):** Breathtaking hover-cards detail the three key pillars: Fabric Innovation, Premium Detailing, and Functional Benefit.
4. **Chameleon Style Guide ("Your Personal Stylist"):** A tabbed showcase representing the 6 styling modes (Weekend Chill, City Smart, Work Mode, Date Night, Vacation Ready, Gym to Go) with styling tips and an adapting mini chameleon preview.
5. **Endless Avatars Showcase:** Social post cards displaying the weekly social media avatars (The Singer, Dancer, Comedian, News Anchor).
6. **Teaser Sandbox:** Interactive simulators for:
   - *Trial Room Shifter:* Click-to-toggle trial room doors showing peeking chameleon tails and shifting neon backlights.
   - *The Desire Meter:* A click-and-drag tongue slider reaching out from high above to catch the P365 shipping crate.

## How to Run & View

You can open the project in two ways:

### Option 1: Direct File Opening
Double-click the [index.html](file:///C:/Users/OrCon/.gemini/antigravity/scratch/benetton-p365-campaign/index.html) file inside this folder to open the page directly in any modern web browser.

### Option 2: Run Local Server (Recommended for perfect asset loading)
To host the files locally using Python:
1. Open terminal in this directory.
2. Run:
   ```bash
   python -m http.server 8000
   ```
3. Open `http://localhost:8000` in your web browser.
